## 2020-01-03

### View 개요

- XML

  - 안드로이드에서 view를 그리는데 사용되는 언어

- view component

  - 안드로이드 화면을 구성하는 요소들
  - Textview, imageview 등등

- View component의 종류

  - 부모가 될 수 있는 view component(자식의 배치조정)
    - Linear Layout
    - RelativeLayout
    - FrameLayout
    - ScrollView
    - 등등
  -  자식이 될 수 있는 view component
    - Textview, imageview 등등 모든 view component

- 속성(Attribute)

  - 모든 view component가 가지고 있는 속성
    - width
    - height
    - background
  - 각각의 view component가 가지고 있는 속성이 있음

- 화면 크기 단위

  - px - 절대 크기 나타냄 , 물리적 크기와 상관없이 나타남
  - dp - 디바이스화면 대비 크기 , 가능한 dp 를 사용

- 실습

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
      xmlns:app="http://schemas.android.com/apk/res-auto"
      xmlns:tools="http://schemas.android.com/tools"
      android:layout_width="match_parent"
      android:layout_height="match_parent"
      tools:context=".MainActivity">
  
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorAccent"
          android:text="안드로이드"
          />
  </LinearLayout>
  ```

  

### View 크기

- view 크기

  - 숫자로 표시

  - WrapContent
    -  view 내용물 크기만큼 크기를 설정
  - MatchParent
    - 부모의 크기만큼 설정

- padding /  margin

  - margin
    - 자식 뷰를 부모 뷰로 부터 얼마나 떨어트려 놓을건지
  - padding
    - 해당 뷰 컴포넌트로 부터 내용물을 얼마나 떨어트려 놓을건지

- 실습

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
      xmlns:app="http://schemas.android.com/apk/res-auto"
      xmlns:tools="http://schemas.android.com/tools"
      android:layout_width="match_parent"
      android:layout_height="match_parent"
      tools:context=".MainActivity"
      android:background="@color/colorPrimary">
  
      <TextView
          android:layout_width="match_parent"
          android:layout_height="wrap_content"
          android:background="@color/colorAccent"
          android:text="안녕하세요"
          android:textSize="30dp"/>
  
  </LinearLayout>
  ```

### View - Linearlayout

- LinearLayout

  - 가지고 있는 자식 들의 배치를 설정

  - 주요 속성

    - orientation 

      -가로 방향으로 배치할 것인지 새로 방향으로 배치할 것인지 설정

    - weight(자식에서 사용)

      -자식 뷰의 크기를 가중치에 의해서 결정

    - Gravity (부모에서 사용)

      -치우침

      -자식 뷰를 부모의 어떤 방향으로 치우치게 배치할 것인지 설치

- 실습

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <!--최상위 뷰이자 부모 뷰-->
  <LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
      xmlns:app="http://schemas.android.com/apk/res-auto"
      xmlns:tools="http://schemas.android.com/tools"
      android:layout_width="match_parent"
      android:layout_height="match_parent"
      android:orientation="vertical"
      tools:context=".MainActivity">
  
      <!--자식 뷰들-->
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimaryDark" />
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimary" />
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorAccent"/>
  
  </LinearLayout>
  ```

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
      android:orientation="horizontal" android:layout_width="match_parent"

      <TextView
  ```

### View - RelativeLayout

- RelativeLayout

  - 자식 뷰의 배치를 상대적으로 조정

- 기준을 정하는 방법

  1. RelativeLayout 자체를 기준으로 사용하여 자식 뷰를 배치
     - alignParent ~
  2. 기준을 지정하여 그 기준 기본으로 자식 뷰를 배치
     - top, right, below , left

- 실습

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
      xmlns:app="http://schemas.android.com/apk/res-auto"
      xmlns:tools="http://schemas.android.com/tools"
      android:layout_width="match_parent"
      android:layout_height="match_parent"
      tools:context=".MainActivity">
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimaryDark"
          android:layout_alignParentRight="true"/>
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimaryDark"
          android:layout_alignParentBottom="true"/>
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimaryDark"
          android:layout_alignParentTop="true"/>
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimaryDark"
          android:layout_alignParentRight="true"
          android:layout_alignParentBottom="true"/>
  
      <!--부모의 가운대로 자식 뷰가 위치-->
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimaryDark"
          android:layout_centerInParent="true"/>
  
  
  </RelativeLayout>
  ```

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
      android:orientation="horizontal" android:layout_width="match_parent"
      android:layout_height="match_parent">
  
      <!--기준뷰에 아이디를 지정해서 기준으로 지정-->
      <TextView
          android:id="@+id/gijun"
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorPrimaryDark"
          android:layout_centerInParent="true"/>
      <!--기준 뷰의 오른쪽으로 이동-->
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorAccent"
          android:layout_toRightOf="@+id/gijun"/>
      <!--기준 뷰의 왼쪽으로 이동-->
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorAccent"
          android:layout_toLeftOf="@+id/gijun"/>
      <!--기준 뷰의 위쪽으로 이동-->
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorAccent"
          android:layout_above="@+id/gijun"/>
      <!--기준 뷰의 아래쪽으로 이동-->
      <TextView
          android:layout_width="100dp"
          android:layout_height="100dp"
          android:background="@color/colorAccent"
          android:layout_below="@+id/gijun"/>
  </RelativeLayout>
  ```

  

### View - FrameLayout

- FrameLayout

  - 뷰 컴포넌트들을 겹쳐 보이고 싶을 때
  - gravity 속성
    - Left | top
    - center_horizontal | top
    - Rgiht | bottom

- 실습

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
      xmlns:app="http://schemas.android.com/apk/res-auto"
      xmlns:tools="http://schemas.android.com/tools"
      android:layout_width="match_parent"
      android:layout_height="match_parent"
      tools:context=".MainActivity">
      <!--먼저 적어준게 가장 아래층에 위치한다.-->
      <LinearLayout
          android:layout_width="match_parent"
          android:layout_height="match_parent"
          android:background="@color/colorAccent">
      </LinearLayout>
      <LinearLayout
      android:layout_width="300dp"
      android:layout_height="300dp"
      android:background="@color/colorPrimaryDark">
      </LinearLayout>
      <LinearLayout
          android:layout_width="200dp"
          android:layout_height="200dp"
          android:background="@color/colorPrimary">
      </LinearLayout>
  </FrameLayout>
  ```

### View - ScrollView

- scrollview

  - 내가 지정한 뷰의 영역보다 내용물이 많을 때 사용

  - 오직 하나의 자식 뷰만을 가질 수 있다.
  - fillViewport 속성
  - scrollbars 속성
    - none , Vertical , Horizontal

- 실습

  ```xml
  <?xml version="1.0" encoding="utf-8"?>
  <LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
      xmlns:app="http://schemas.android.com/apk/res-auto"
      xmlns:tools="http://schemas.android.com/tools"
      android:layout_width="match_parent"
      android:layout_height="match_parent"
      android:orientation="vertical"
      tools:context=".MainActivity">
      <!-- 자식뷰를 match_parent를 해도 꽉차지 않을 때 fillviewport를 true로 설정-->
      <ScrollView
          android:layout_width="match_parent"
          android:fillViewport="true"
          android:scrollbars="vertical"
          android:layout_height="match_parent">
          <LinearLayout
              android:layout_width="match_parent"
              android:layout_height="match_parent"
              android:orientation="vertical"
              android:background="@color/colorAccent">
              <TextView
                  android:layout_width="300dp"
                  android:layout_height="300dp"
                  android:layout_marginBottom="16dp"
                  android:background="@color/colorPrimaryDark"></TextView>
              <TextView
                  android:layout_width="300dp"
                  android:layout_height="300dp"
                  android:layout_marginBottom="16dp"
                  android:background="@color/colorPrimaryDark"></TextView>
              <TextView
                  android:layout_width="300dp"
                  android:layout_height="300dp"
                  android:layout_marginBottom="16dp"
                  android:background="@color/colorPrimaryDark"></TextView>
          </LinearLayout>
      </ScrollView>
  </LinearLayout>
  ```
